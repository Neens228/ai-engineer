# Task CLI — менеджер задач в командной строке

Простое консольное приложение для управления списком задач с сохранением в базе данных SQLite.
Первый учебный проект на пути к AI Engineer: чистая архитектура, типизация и работа с БД.

## Возможности
- Добавление задач
- Просмотр списка задач со статусом выполнения
- Отметка задачи выполненной
- Постоянное хранение в SQLite (задачи не теряются между запусками)

## Стек
- **Python 3.13**
- **Pydantic** — модель данных и валидация
- **SQLite** (модуль `sqlite3`) — хранение
- **uv** — управление окружением и зависимостями

## Архитектура
Приложение разделено на слои по ответственности (separation of concerns):

| Файл | Слой | Ответственность |
|------|------|-----------------|
| `models.py` | Модель | структура задачи (`Task`) |
| `storage.py` | Хранилище | работа с SQLite (CRUD), отдаёт чистые модели |
| `main.py` | Интерфейс | разбор команд пользователя, цикл приложения |

Каждый слой знает только о слое ниже: `main` → `storage` → `models`.

## Запуск
```bash
# из папки проекта
python main.py
```

## Команды
| Команда | Действие |
|---------|----------|
| `add <текст>` | добавить задачу |
| `list` | показать все задачи |
| `done <номер>` | отметить задачу выполненной |
| `exit` | выйти |

### Пример сессии
```
> add Купить молоко
Задача #1 добавлена.
> add Помыть посуду
Задача #2 добавлена.
> list
[ ] 1. Купить молоко
[ ] 2. Помыть посуду
> done 1
Задача #1 выполнена.
> list
[x] 1. Купить молоко
[ ] 2. Помыть посуду
> exit
Пока!
```
